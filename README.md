# :brazil: braza-candidatos-wordcloud

Nuvem de palavras do Plano de Governo dos candidatos à Presidência da República. Foi gerada também uma nuvem de palavras geral, unindo o texto dos planos de governo de todos os candidatos. 

Quanto maior a palavra, maior a frequência em que ela aparece no texto.

Planos de Governo retirados diretamente do site oficial da campanha dos candidatos.

**Bibliotecas utilizadas**

- swig  (`apt install swig`)
- textract (`pip install textract`)
- wordcloud (`pip install wordcloud`)
- matplotlib (`pip install matplotlib`)
